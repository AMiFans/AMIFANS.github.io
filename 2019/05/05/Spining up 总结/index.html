Spinning Up

2019-05-05 | 深度强化学习

主要算法

On-Policy算法

  • 香草策略梯度（VPG）
  • 信托区域策略优化（TRPO）
  • 近端策略优化（PPO）

VPG 是深度 RL 空间中最基本的入门级算法，因为它完全超越了 DRL 的出现。VPG 的核心要素一直追溯到80年代末/ 90年代初。它开始了一系列的研究，最终导致了更强大的算法，如TRPO，然后很快PPO。

这一系列工作的一个关键特性是所有这些算法都是 On-Policy 的：即，它们不使用旧数据，这使得它们在样本效率方面很弱。但这是有充分理由的：这些算法直接优化了我们关心的目标 - 策略性能 - 并且数学上需要您在策略数据上计算更新。因此，这一类算法可以降低样本效率，有利于稳定性 - 但您可以看到技术的进展（从VPG到TRPO再到PPO），以弥补样本效率的不足。

Off-Policy算法

  • 深度确定性策略梯度（DDPG）
  • 双延迟DDPG（TD3）
  • Soft Actor-Critic（SAC）

DDPG 是类似于 VPG 的基础算法，导致DDPG的DPG理论直到2014年才发表。DDPG 与 Q-learning 算法密切相关，并且它同时学习Q-函数和策略，这两者的更新可以相互促进对方。

像 DDPG 和 Q-Learning这样的算法是Off-Policy的，因此它们能够非常有效地重用旧数据。他们通过利用Bellman方程的最优性来获得这种好处，可以训练Q函数以满足使用任何环境交互数据（只要有足够的经验来自环境中的高回报区域）。

但问题是，无法保证很好地满足贝尔曼方程式会带来很好的策略性能。根据经验，人们可以获得很好的性能 - 当它发生时，样本效率很好 - 但缺乏保证会使这类算法变得脆弱和不稳定。TD3和SAC是DDPG的后代，它们利用各种见解来缓解这些问题。
