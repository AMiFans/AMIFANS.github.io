伯努利分布

　　伯努利分布，又称 0-1 分布。对于一个伯努利试验，它只有两个结果 1 或 0。通常，取1的概率为 p，取 0 的概率为 1−p。p 就是该伯努利分布的参数。对于随机变量 X 服从伯努利分布，$P_r(X=1)=p$, $P_r(X=0)=q=1−p$, $0<p<1$。可以发现，对于伯努利分布，本来是需要两个参数的 p 和 q 的，但是由于 p 和 q 相关，只有一个自由度。所以一般用 p 来表示就足够了。相应的概率质量函数（pmf）为：
